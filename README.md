## `> cat euler-formula.md`

In graph theory, elegance is everything. **Euler's Formula** for planar graphs — no matter how complex a network gets, as long as no edges cross, the relationship between Vertices (V), Edges (E), and Faces (F) remains perfectly unbroken:

$$V - E + F = 2$$
